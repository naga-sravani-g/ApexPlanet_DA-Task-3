This project focuses on performing a deep-dive analysis of Amazon sales data to solve complex business problems. The primary goal was to define key performance indicators (KPIs) and visualize customer retention through Cohort Analysis.

📊 Key Deliverables
Core KPIs Defined: * Average Order Value (AOV): Measuring the average spend per transaction.

Retention Rate: Analyzing the percentage of repeat customers.

Deep-Dive Analysis: A comprehensive Cohort Analysis to track customer retention over time, identifying patterns in user loyalty and acquisition.

Interactive Visualization: An automated dashboard (built with Python/Plotly) that allows users to explore retention trends across different months.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn/Plotly

Tooling: Jupyter Notebook / Google Colab

📈 Key Insights
Identified the specific months where customer drop-off is highest.

Calculated synthetic Customer Lifetime Value trends to guide marketing re-engagement strategies.
